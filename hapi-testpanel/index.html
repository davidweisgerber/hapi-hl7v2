HAPI TestPanel

SOCIALPLUGINSHERETP

The HAPI TestPanel is a full featured HL7 message editor, transmitter and receiver. This software is still an alpha release, so please use it with caution!

TestPanel is free software, distributed under the MPL/GPL. TestPanel is also a demonstration of the many features of HAPI, the best HL7 programming API available for Java.
